Help & Documentation

Overview

The Antimicrobial Usage Alerts module monitors broad-spectrum antibiotic duration for hospitalized patients. When a monitored medication exceeds the configured threshold (default: 72 hours), an alert is generated for ASP pharmacist review.

The goal is to prompt timely de-escalation conversations, reduce unnecessary broad-spectrum exposure, and support antimicrobial stewardship best practices.

Monitored Medications

Medication RxNorm Code Common Indications
Meropenem 29561 Broad-spectrum carbapenem; multi-drug resistant gram-negative infections, intra-abdominal infections, meningitis
Vancomycin 11124 MRSA coverage; bacteremia, endocarditis, osteomyelitis, CNS infections

Duration Thresholds

Threshold Duration Severity Action
Standard ≥ 72 hours (3 days) {% include "_includes/badge.html" with text="High" color="amber" %} Review for potential de-escalation based on culture data
Urgent ≥ 144 hours (6 days) {% include "_includes/badge.html" with text="Critical" color="red" %} Urgent review — consider de-escalation or discontinuation

The 72-hour threshold aligns with the typical 48-72 hour window for culture results to return, after which empiric broad-spectrum therapy should be reassessed.

Alert Severity Levels

Severity Condition Description
{% include "_includes/badge.html" with text="Critical" color="red" %} ≥ 144 hours (2x threshold) Extended broad-spectrum use requiring urgent review for de-escalation or discontinuation
{% include "_includes/badge.html" with text="High" color="amber" %} ≥ 72 hours (threshold) Broad-spectrum use exceeding review threshold; culture results should be available

Alert Workflow

  1. Alert Created — System detects medication exceeding threshold
  2. Acknowledge — ASP pharmacist acknowledges review of the alert
  3. Resolve — Document the clinical decision with a resolution reason

Resolution Options

Resolution When to Use
Therapy Changed / De-escalated Antibiotic changed to narrower-spectrum agent based on culture/clinical data
Therapy Stopped Antibiotic discontinued (course complete, not indicated, etc.)
Clinical Justification Continued broad-spectrum use is clinically justified (e.g., polymicrobial infection)
Discussed with Team Discussed de-escalation options with primary team
Messaged Team Sent recommendation to primary team via EHR messaging
Culture Pending Awaiting culture results before making a change
Escalated to Attending Referred to attending physician or ID consultant
No Action Needed Alert reviewed, no change required at this time
Patient Discharged Patient has been discharged
False Positive Alert triggered in error (e.g., medication already discontinued)

ASP De-escalation Workflow

  1. Review the alert and confirm the medication is still active
  2. Check culture and sensitivity results in the EHR
  3. If cultures show susceptible organism: recommend targeted (narrower-spectrum) therapy
  4. If cultures are negative: consider whether antibiotics are still indicated
  5. If cultures are pending: consider clinical trajectory and reassess when results return
  6. Document the recommendation and resolution in AEGIS
